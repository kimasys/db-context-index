# Contributing

Thank you for contributing to db-context-index.

This repository focuses on improving how AI coding agents work with PostgreSQL, Supabase, SQL migrations, RLS policies, and database context.

## Goals

Contributions should improve:

- token efficiency
- migration safety
- context retrieval
- schema indexing
- RLS understanding
- dependency mapping

## Guidelines

- Prefer clarity over abstraction.
- Prefer additive changes.
- Avoid unnecessary complexity.
- Keep examples realistic.
- Focus on practical engineering value.
- Keep documentation readable for both humans and AI agents.

## Commit examples

```txt
feat: add migration risk mapping example
fix: improve RLS checklist
docs: add token efficiency section
```
